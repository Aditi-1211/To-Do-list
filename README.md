**To-Do List Web App (Frontend)

A simple and clean To-Do List web application built using HTML, CSS, and JavaScript.
This project helps users add, view, mark, and delete daily tasks with an intuitive interface.

--------------------------------------------------------------------------------------------------------------------------

**Features

-Add new tasks easily

-Mark tasks as completed

-Delete tasks

-Filter tasks by All / Completed / Pending

-Minimal and clean UI

-----------------------------------------------------------------------------------------------------------------------------

**Tech Stack

-HTML5 – page structure

-CSS3 – styling and layout

-JavaScript (ES6) – dynamic behavior & functionality
